# Portfolio

Statisches Portfolio von Alireza Esmaeili mit Deutsch/English Umschalter und Darkmode.

## Dateien

- `index.html` - Seitenstruktur
- `style.css` - Layout, responsive Design und Darkmode
- `script.js` - Inhalte, Sprache, Theme und Rendering
- `images/` - Profilbild und Projektbilder

## Lokal starten

`index.html` im Browser öffnen.

## GitHub Pages

Die Seite läuft als statische GitHub-Pages-Seite ohne Admin-Bereich, ohne Build-Schritt und ohne externe Datenbank.
